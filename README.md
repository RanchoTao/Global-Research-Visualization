# Global-Research-Visualization
全世界范围内的科研水平和进度可视化Visualization of global scientific research levels and progress

# 学术数据库
Web of Science、Scopus、PubMed、arXiv、CNKI、Microsoft Academic...

# 爬虫方案
Scrapy:大规模分布式数据抓取
（要是能解决芯片问题就好了...)

# 数据清洗
MySQL，算法自动清洗


# 研究主题识别
文本挖掘，基于词向量的语义建模

# 衡量科研水平与进度
量化指标：进度、科技树、热度、引用次数

# 识别前沿与趋势
分析关键词再时间序列上的出现频率突变可以识别新兴趋势和前沿领域


# 技术尝试
前端：JavaScript、Gephi、React/Vue.js
后端：Python
